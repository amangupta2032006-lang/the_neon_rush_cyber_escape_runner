# Neon Rush: Cyber Scape Runner

A high-speed, browser-based endless runner game featuring a vibrant, neon-drenched cyberpunk aesthetic. Players navigate a futuristic cityscape, dodging obstacles while the game dynamically increases in difficulty.

## 🚀 Project Overview
**Neon Rush** was developed to explore high-performance web animations and responsive game design. The project focuses on real-time rendering, collision detection logic, and creating an immersive arcade experience using native web technologies.

## 🛠 Technical Stack
* **Core:** HTML5, CSS3, JavaScript (ES6+)
* **Graphics:** Canvas API / CSS-based animations
* **Design Philosophy:** Responsive UI, Cyberpunk color palette, and optimized game loop logic

## 🎮 Key Features
* **Dynamic Difficulty:** Obstacle spawning frequency and movement speed adjust in real-time.
* **Responsive Gameplay:** Optimized for both desktop and mobile web browsers.
* **Immersive Visuals:** Use of neon gradients, keyframe animations, and high-contrast UI elements.
* **Performance Focused:** Lightweight build with zero external framework dependencies.
